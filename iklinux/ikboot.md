# ikboot — управление загрузчиком

Упрощает работу с загрузчиком в [[IrekKom Linux]].
Снижает порог входа для пользователей, которые боятся трогать boot-раздел.

---

## Команды

| Команда | Описание |
|---|---|
| `ikboot status` | Текущий загрузчик и записи |
| `ikboot use <загрузчик>` | Переключиться на GRUB / systemd-boot / rEFInd |
| `ikboot add <запись>` | Добавить запись загрузки |
| `ikboot remove <запись>` | Удалить запись загрузки |
