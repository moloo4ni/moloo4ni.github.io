# ikwiki — вики прямо в TTY

Позволяет читать документацию [[IrekKom Linux]] не выходя из терминала.
Полезно при установке системы без графического окружения.

---

## Команды

| Команда | Описание |
|---|---|
| `ikwiki <запрос>` | Поиск по вики |
| `ikwiki install` | Руководство по установке дистрибутива |
