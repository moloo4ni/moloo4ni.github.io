# ikinstall — установщик IK Linux

Устанавливает [[IrekKom Linux]] в режиме TUI — без необходимости
вводить команды вручную. Альтернатива мануальной установке.

В процессе установки настраивается:
- файловая система и разметка диска
- init-система
- shell
- канал обновлений (stable / rolling)
- триггеры снапшотов
- промпт fish (опционально)

---

## Запуск

| Команда     | Описание             |
| ----------- | -------------------- |
| `ikinstall` | Запустить установщик |
