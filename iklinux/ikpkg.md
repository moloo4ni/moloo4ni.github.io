# ikpkg — менеджер пакетов IK Linux

Декларативный пакетный менеджер. Состояние системы описывается 
в файле `packages.toml` — ikpkg приводит систему в соответствие с ним.

---

## Команды

| Команда | Описание |
|---|---|
| `ikpkg -i <пакет>` | Установить пакет |
| `ikpkg -r <пакет>` | Удалить пакет |
| `ikpkg -u` | Обновить все пакеты |
| `ikpkg -u <пакет>` | Обновить конкретный пакет |
| `ikpkg sync` | Привести систему в соответствие с `packages.toml` |
| `ikpkg diff` | Показать что изменится перед sync |
| `ikpkg rollback` | Откат к предыдущему состоянию |

---

## Каналы обновлений

Выбирается при установке:
- **stable** — стабильные релизы
- **rolling** — свежие пакеты, обновления сразу после выхода

Переключить канал можно через `ikpkg` в любой момент.
