# ikconf — быстрый доступ к конфигам

Открывает конфиг любого установленного пакета в выбранном редакторе. 
Расположение конфигов берётся из манифеста пакета — искать вручную не нужно.

---

## Команды

| Команда | Описание |
|---|---|
| `ikconf edit <пакет>` | Открыть конфиг пакета в редакторе |
| `ikconf list` | Показать все доступные конфиги |
| `ikconf path <пакет>` | Показать путь к конфигу без открытия |
