# ikfs — утилита для работы с файловыми системами

Единая обёртка для работы с любой ФС в [[IrekKom Linux]].

---

## Команды

| Команда | Описание |
|---|---|
| `ikfs status` | Текущая ФС, точки монтирования, использование |
| `ikfs convert btrfs` | Помощник миграции с ext4 на btrfs |
| `ikfs snapshot` | Создать снапшот (btrfs/zfs) |
| `ikfs check` | Проверка ФС (обёртка fsck) |

---

## Снапшоты

Триггер создания снапшотов настраивается пользователем:
- вручную
- при установке конкретных пакетов
- при установке любого пакета
