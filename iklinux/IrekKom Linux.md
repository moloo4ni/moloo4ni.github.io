# IrekKom Linux (IK Linux)

Декларативный дистрибутив с идеей **выбора без ограничений**. 
Минимум по умолчанию — максимум свободы настройки. 
Никакой телеметрии. Репозитории доступны из любой страны.

---

## Компоненты по умолчанию

| Компонент | По умолчанию | Альтернативы |
|---|---|---|
| Init | dinit | s6, runit |
| Пакетный менеджер | [[ikpkg]] | — |
| Файловая система | ext4 | btrfs, zfs и др. |
| Shell | fish | bash, zsh, ksh и др. |
| DE/WM | — | на выбор пользователя |
| База | собственная | — |

---

## Системные утилиты

- [[ikpkg]] — декларативный менеджер пакетов
- [[ikfs]] — работа с файловыми системами
- [[ikwiki]] — чтение вики прямо в TTY
- [[ikinstall]] — установщик в режиме TUI
- [[iknet]] — настройка сети
- [[ikboot]] — управление загрузчиком
- [[ikconf]] — быстрый доступ к конфигам любого пакета

---

## Установка

После загрузки система встречает пользователя **TTY** — никакого навязанного окружения.

Установка возможна двумя способами:
- **Мануально** — с помощью [[ikwiki]] или любого устройства с интернетом
- **TUI** — с помощью [[ikinstall]]
