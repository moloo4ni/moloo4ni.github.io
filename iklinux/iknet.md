# iknet — настройка сети

Единая утилита для управления сетевыми менеджерами в [[IrekKom Linux]].
Позволяет выбрать и переключиться между менеджерами без ручной настройки.

---

## Команды

| Команда                | Описание                                       |
| ---------------------- | ---------------------------------------------- |
| `iknet status`         | Текущее состояние сети и активный менеджер     |
| `iknet use <менеджер>` | Переключиться на NetworkManager / iwd / dhcpcd |
| `iknet connect <сеть>` | Подключиться к сети                            |
| `iknet list`           | Список доступных сетей                         |
